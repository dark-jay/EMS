# EMS
A simple EMS (Employee Management System) written in PHP Codeigniter

### All functionalities of CRUD operations have been inculded.
* User registration page
* User login page
* Add Employee page
* Employee List
* Add Job
* Jobs List

### Versions
* PHP 7.4.26
* CodeIgniter 3.1.11
* XAMP 3.3.0
